# 성장기 어린이 키 예측 AI 모델 - 기능 명세서

## 1. 프로젝트 개요

### 1.1 목적
성장기 어린이의 현재 키와 관련 정보를 입력받아, 머신러닝 모델을 기반으로 예상 최종 키(성인 키)를 예측하는 인공지능 시스템 개발

### 1.2 주요 목표
- 성장기 어린이의 현재 키를 기반으로 예상 성인 키 예측
- 데이터 기반의 과학적인 예측 모델 제공
- 사용자 친화적인 인터페이스 제공

---

## 2. 주요 기능

### 2.1 데이터 입력
사용자로부터 다음 정보를 입력받습니다:
- **필수 입력값**:
  - 현재 키 (cm)
  - 나이 (세 또는 월령)
  - 성별 (남/여)
  
- **선택 입력값** (정확도 향상을 위해):
  - 현재 체중 (kg)
  - 부모 키 (아버지 키, 어머니 키)
  - 생년월일 (정확한 월령 계산용)

### 2.2 데이터 검증
- 입력값의 유효성 검사 (범위 확인, 타입 확인)
- 비정상적인 값에 대한 경고 메시지
- 필수 입력값 누락 시 오류 처리

### 2.3 예측 모델
- 머신러닝 회귀 모델 적용 (선형 회귀, 랜덤 포레스트, XGBoost 등)
- 성별별, 연령별 성장 패턴 고려
- 부모 키 정보를 활용한 유전적 요인 반영

### 2.4 예측 결과 출력
- 예상 최종 키 (cm)
- 예측 신뢰 구간 (예: ±3cm)
- 현재 성장 곡선 상 위치 (백분위수)
- 성장 추세 분석

### 2.5 시각화
- 성장 곡선 그래프 (현재 위치와 예측 경로 표시)
- 성별/연령별 평균 키와의 비교
- 성인 키 예측 분포 그래프

## 7. 성능 평가 지표

### 7.1 회귀 모델 평가 지표
- **MAE (Mean Absolute Error)**: 평균 절대 오차
- **RMSE (Root Mean Square Error)**: 평균 제곱근 오차
- **R² Score**: 결정 계수
- **MAPE (Mean Absolute Percentage Error)**: 평균 절대 백분율 오차

### 7.2 교차 검증
- K-Fold Cross Validation (K=5 또는 10)
- 성별별, 연령별 성능 평가

## 9. 주의사항 및 제한사항

### 9.1 의료적 면책 조항
- 이 모델의 예측 결과는 참고용이며, 의료적 조언을 대체할 수 없습니다.
- 실제 성장 장애나 건강 문제가 의심될 경우 전문의 상담이 필요합니다.

### 9.2 데이터 제한사항
- 학습 데이터의 품질과 양에 따라 예측 정확도가 달라질 수 있습니다.
- 개인의 유전적 특성, 생활 환경, 영양 상태 등 다양한 요인이 성장에 영향을 미칩니다.

### 9.3 모델 제한사항
- 일반적인 성장 패턴을 기반으로 한 예측이므로, 개인차가 있을 수 있습니다.
- 성장기 종료 후 예측은 의미가 없습니다.
