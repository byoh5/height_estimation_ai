# 오프셋 조절 기능 가이드

## 📊 개요

키 예측의 정확도를 향상시키기 위해 다양한 보정 기능을 제공합니다:

1. **국가/지역별 보정**: 국가별 평균 키 차이 반영
2. **유전 공식 보정**: 다양한 유전 예측 공식 적용
3. **개인 성장 패턴 분석**: 과거 키 기록 기반 성장 패턴 분석

### 작동 원리
국가별 평균 키와 기본 평균 키의 비율을 계산하여 예측값에 적용합니다.

```
보정된 예측 = 기본 예측 × (국가 평균 / 기본 평균)
```

## 🧬 유전 공식 보정

### 1. Mid-Parental Height 공식
가장 기본적인 유전 공식

```
남아: (아버지 키 + 어머니 키 + 13) / 2
여아: (아버지 키 + 어머니 키 - 13) / 2
```

### 2. Tanner 공식 (Tanner et al., 1970)
더 정확한 유전 예측 공식

```
남아: (아버지 키 + 어머니 키) / 2 + 6.5
여아: (아버지 키 + 어머니 키) / 2 - 6.5
```

### 3. Khamis-Roche 공식
현재 키와 나이를 고려한 예측 공식

성장 잠재력을 나이에 따라 조정하여 예측합니다.

### 적용 방식
세 가지 공식의 예측값을 평균 내어 기본 예측과 가중 평균합니다:
- 기본 예측: 70%
- 유전 공식 평균: 30%

## 📈 개인 성장 패턴 분석

### 성장 속도 분석
과거 키 기록을 분석하여 연간 성장률을 계산합니다.

### 패턴 분류
- **정상 (normal)**: 연간 3-8cm 성장
- **느림 (slow)**: 연간 3cm 미만
- **빠름 (rapid)**: 연간 8cm 초과

### 보정 적용
- 느린 성장 패턴: 예측값 × 0.97 (3% 감소)
- 빠른 성장 패턴: 예측값 × 1.03 (3% 증가)
- 정상 패턴: 보정 없음

## 🔧 사용 방법

### Python 코드

```python
from src.modeling.enhanced_predictor import EnhancedHeightPredictor

predictor = EnhancedHeightPredictor()

result = predictor.predict(
    birth_date='2015-01-15',
    gender='M',
    current_height_cm=140,
    father_height_cm=175,
    mother_height_cm=162,
    country_code='KR',  # 국가 보정
    use_genetic_formulas=True,  # 유전 공식 사용
    use_growth_pattern=True,  # 성장 패턴 분석 사용
    height_history=[
        {'date': '2020-01-15', 'height_cm': 120},
        {'date': '2022-01-15', 'height_cm': 130}
    ]
)

print(f"예측 키: {result['predicted_height']:.1f}cm")
print(f"기본 예측: {result['details']['original_prediction']:.1f}cm")
print(f"총 보정: {result['details']['total_adjustment']:+.2f}cm")
print(f"보정 항목: {list(result['details']['adjustments'].keys())}")
```

## ⚙️ 보정 옵션

### use_genetic_formulas
- `True`: 유전 공식 보정 적용 (부모 키가 있을 때만)
- `False`: 유전 공식 보정 미적용

### use_growth_pattern
- `True`: 개인 성장 패턴 분석 적용 (과거 기록이 2개 이상일 때만)
- `False`: 성장 패턴 분석 미적용

### country_code
- 국가 코드 (KR, JP, CN, US 등)
- 'DEFAULT': 국가 보정 미적용

## 📝 참고 사항

1. **국가 보정**은 항상 적용됩니다 (국가 선택 시)

3. **성장 패턴 분석**은 과거 키 기록이 2개 이상일 때만 적용됩니다
4. 보정은 누적 적용됩니다 (국가 → 유전 → 성장 패턴 순서)
