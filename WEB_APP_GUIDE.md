```bash

```

---

## 📱 기능 설명

### 성인 키 예측 탭

#### 입력 항목
- **성별** (필수): 남아 또는 여아 선택
- **현재 나이** (선택): 세 단위
- **현재 키** (선택): cm 단위
- **아버지 키** (선택): cm 단위
- **어머니 키** (선택): cm 단위

#### 결과 표시
- 예상 성인 키 (큰 숫자로 표시)
- 신뢰도 배지 (매우 높음/높음/보통/낮음)
- 사용된 모델 정보
- 각 모델별 예측 값 (부모 키 모델, 성장 패턴 모델)

#### 특징
- 입력 정보가 많을수록 더 정확한 예측
- 부모 키 정보가 있으면 매우 높은 신뢰도
- 두 모델을 결합한 가중 평균 사용

---

### 성장 곡선 탭

#### 입력 항목
- **성별** (필수): 남아 또는 여아 선택
- **현재 나이** (필수): 세 단위
- **현재 키** (필수): cm 단위

#### 결과 표시
- **성장 곡선 그래프**: 시각적 차트로 여러 나이의 키 표시
- **예측 결과 표**: 각 나이별 예상 키와 성장량 표

#### 예측 가능한 나이
- 5세
- 10세
- 15세
- 18세

## 🔌 API 엔드포인트

### 1. 성인 키 예측
```
POST /api/predict/adult
Content-Type: application/json

{
  "gender": "M",
  "age_years": 10,
  "height_cm": 140,
  "father_height_cm": 175,
  "mother_height_cm": 162
}
```

### 2. 성장 곡선 예측
```
POST /api/predict/growth-curve
Content-Type: application/json

{
  "gender": "M",
  "current_age_years": 8,
  "current_height_cm": 130
}
```

### 3. 특정 나이 키 예측
```
POST /api/predict/age
Content-Type: application/json

{
  "gender": "M",
  "current_age_years": 8,
  "current_height_cm": 130,
  "target_age_years": 15
}
```

### 4. 사용 가능한 예측 나이 목록
```
GET /api/available-ages
```

## 🐛 문제 해결

### 모델이 로드되지 않는 경우
- `models/saved_models/` 디렉토리에 모델 파일이 있는지 확인
- 모델 학습 스크립트를 먼저 실행

### 포트가 이미 사용 중인 경우
- `app.py`의 `port` 값을 다른 값(예: 5001)으로 변경

### 예측 오류가 발생하는 경우
- 입력 데이터 형식 확인
- 필수 항목 입력 확인
- 서버 로그 확인
