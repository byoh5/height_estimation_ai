### 2. Parents' Heights vs Adult Children's Heights (archive.zip) ⭐⭐⭐⭐⭐

#### 기본 정보
- **원본 파일**: `GaltonFamilies.csv`
- **크기**: 0.03 MB
- **행 수**: **934행**
- **컬럼 수**: 9개

#### 데이터 구조
```
- rownames: 행 번호
- family: 가족 ID
- father: 아버지 키 (인치)
- mother: 어머니 키 (인치)
- midparentHeight: 중간 부모 키 (인치)
- children: 가족 내 자녀 수
- childNum: 가족 내 자녀 번호
- gender: 성별 (male/female)
- childHeight: 자녀 키 (인치) - 성인 키
```

#### 통계 정보 (인치 → cm 변환 필요)
- **부모 키** (인치):
  - 아버지: 평균 69.2인치 (175.8cm), 표준편차 2.5인치
  - 어머니: 평균 64.1인치 (162.8cm), 표준편차 2.3인치
  - 중간 부모 키: 평균 69.2인치 (175.8cm)
- **자녀 키** (인치):
  - 평균: 66.7인치 (169.5cm)
  - 범위: 56.0~79.0인치 (142.2~200.7cm)
- **성별 분포**:
  - 남아: 481명 (51.5%)
  - 여아: 453명 (48.5%)
- **가족당 자녀 수**: 평균 6.2명

#### 특징
- ✅ 실제 측정 데이터 (1886년 Galton 연구)
- ✅ 부모 키 정보 포함 (유전적 요인 분석 가능)
- ✅ 성별 정보 포함
- ✅ 가족 단위 데이터 (유전 패턴 분석 가능)
- ⚠️ 성인 키 데이터 (성장기 예측에는 부적합할 수 있음)
- ⚠️ 인치 단위 (cm로 변환 필요)
- ⚠️ 나이 정보 없음 (성인 키만 있음)
- ⚠️ 오래된 데이터 (1886년)

#### 활용 방안
- 부모 키와 자녀 키 관계 학습
